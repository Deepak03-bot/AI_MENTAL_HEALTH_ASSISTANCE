# DATA-ANALYTICS-PROJECT
# AI Mental Health Assistant & Data Analytics Internship Project

This repository contains internship tasks, analytics work, and project implementations related to mental health data analysis, psychosocial trend visualization, and AI-assisted healthcare analytics.

## Project Overview

The project focuses on analysing mental health and psychosocial wellbeing indicators using public healthcare and survey datasets. The work includes:

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Mental health trend analysis
- COVID-era impact analysis
- Data visualization and reporting
- Stakeholder insight generation
- AI-assisted healthcare analytics concepts

The repository also includes work related to the **AI Mental Health Assistant** project, which aims to support mental health awareness and data-driven healthcare insights using Artificial Intelligence, Machine Learning, and Data Analytics techniques.

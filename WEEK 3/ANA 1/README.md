# ANA-1 : IAPT Referral-to-Treatment Analysis

This analysis focused on RTT (Referral-to-Treatment) waiting time analysis using NHS Talking Therapies data.

The project calculated mean and median RTT waiting times across different commissioning regions.

## Analysis Performed

- RTT waiting time analysis
- Mean RTT calculation
- Median RTT calculation
- Region-wise comparison
- RTT visualization charts
- Healthcare accessibility analysis

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Files Included

- ANA-1_IAPT_RTT_Analysis.ipynb
- cleaned_rtt_analysis.csv

## Key Findings

- RTT waiting times varied across regions.
- Mean RTT values were generally higher than median RTT values.
- The analysis highlighted differences in regional healthcare accessibility and treatment efficiency.
